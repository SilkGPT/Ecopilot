# Ecopilot

Elixir Implement of copilot

## Installation

For `Darwin`, use `brew install` to quick-start.

```dash
brew install elixir
mix new ecopilot
mix deps.get
```

## Beam and Run

```dash
git clone -j8 --depth 1 -b main git@github.com:SilkGPT/Ecopilot.git eco
cd eco && elixirc lib/ecopilot.ex && elixirc simple.ex
```
